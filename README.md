# 🎮 3FACE - Telegram Bot для организации матчей 5x5 и 2x2

Телеграм бот для организации матчей с друзьями в формате 5x5 и 2x2, аналогичный системе FACEIT.

## ✨ Возможности

- 🎯 **Лобби** - создание лобби на 5 игроков (5x5) или 2 игроков (2x2) для игры с друзьями
- 🔍 **Поиск матча** - автоматический подбор игроков из очереди
- ✅ **Ready Check** - все 10 игроков должны подтвердить готовность перед началом матча
- 🖥️📱 **Разделение по платформам** - отдельные матчи для ПК и мобильных игроков
- ⚔️ **Матчи 5x5** - автоматическое разделение на команды с выбором карты
- 🎲 **Автобаланс команд** - распределение по рейтингу для честной игры
- 👥 **Система пати** - играйте с друзьями в одной команде (независимо от рейтинга)
- 🏆 **Рейтинговая система** - отслеживание побед, поражений и K/D
- 📊 **Топ игроков** - таблица лидеров по рейтингу
- 📷 **Проверка результатов** - модераторы проверяют скриншоты и вносят результаты
- 🛡️ **Защита от багов** - атомарные операции и блокировки предотвращают race conditions
- 🎨 **Красивое оформление** - эмодзи и приятный интерфейс
- 📱 **Mini App** - веб-приложение для просмотра истории матчей и чата с техподдержкой
- 📩 **Система тикетов** - связь с техподдержкой через бота или Mini App

## 🚀 Установка

### 1. Клонирование или скачивание

```bash
cd /путь/к/проекту
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Заполните файл `.env`:

```env
# Telegram Bot Token (получить у @BotFather)
BOT_TOKEN=your_bot_token_here

# ID администраторов (через запятую)
ADMIN_IDS=123456789,987654321

# ID модераторов (через запятую)
MODERATOR_IDS=111111111,222222222
```

### 4. Запуск бота

```bash
python bot.py
```

## 🎮 Как играть

### Для игроков:

1. **Запустите бота** командой `/start`
2. **Пройдите регистрацию:**
   - Введите ваш **игровой ник** (должен совпадать с ником в игре!)
   - Введите ваш **игровой ID** (уникальный идентификатор в игре)
3. **Выберите платформу** (ПК или Телефон) в профиле
4. **Нажмите "🎮 Играть"** в главном меню
5. **Создайте лобби** или присоединитесь к существующему
6. **Дождитесь 10 игроков**
7. **Создатель выбирает карту** - начинается матч
8. **После игры** отправьте скриншот результатов

### Система пати:

1. Нажмите "🎉 Пати" в меню
2. Создайте пати
3. Отправьте ссылку-приглашение друзьям
4. Игроки в пати **всегда** попадают в одну команду!

## 👮 Для модераторов

1. Используйте команду `/admin` для входа в панель
2. Проверяйте скриншоты результатов матчей
3. Выбирайте победителя и вводите счёт
4. Система автоматически обновит рейтинг игроков

### Команды администратора:

- `/admin` - Открыть панель модератора
- `/ban @username` - Забанить игрока
- `/unban @username` - Разбанить игрока
- `/backup` - Создать и скачать бэкап базы данных
- `/addadmin @username` - Назначить администратора
- `/removeadmin @username` - Снять администратора
- `/addmod @username` - Назначить модератора
- `/removemod @username` - Снять модератора
- `/addrating @username 100` - Добавить рейтинг игроку
- `/removerating @username 50` - Снять рейтинг с игрока
- `/setrating @username 1500` - Установить рейтинг игроку

```python
    # ...

```

## 📱 Mini App (Telegram WebApp)

Mini App - это веб-приложение внутри Telegram, которое позволяет:

- **📊 Просматривать историю матчей** - все ваши игры с детальной статистикой
- **💬 Общаться с техподдержкой** - создавать и отслеживать тикеты
- **👤 Видеть профиль** - ваш рейтинг и статистика

### Настройка Mini App

1. **SSL сертификаты** - для работы Mini App требуется HTTPS. Положите сертификаты в папку `3face.xyz/`:
   - `certificate.crt` - сертификат
   - `newkey.key` - приватный ключ

2. **Настройка домена** - укажите в `.env`:
   ```env
   WEBAPP_URL=https://3face.xyz
   WEBAPP_PORT=443
   SSL_CERT_PATH=3face.xyz/certificate.crt
   SSL_KEY_PATH=3face.xyz/newkey.key
   ```

3. **Настройка в BotFather** - установите URL Mini App для вашего бота:
   - Откройте @BotFather
   - Выберите вашего бота
   - Bot Settings → Menu Button → Configure menu button
   - Укажите URL: `https://3face.xyz`

### Кнопка Mini App

В главном меню бота появится кнопка **"📱 Приложение"**, которая открывает Mini App.

## 📝 Примечания

- Стороны (T/CT) меняются после 15 раунда - это учитывается при подсчёте
- Модераторы вручную вводят результаты после проверки скриншотов
- Игроки в пати всегда попадают в одну команду, независимо от рейтинга
- Автобаланс учитывает пати при распределении команд

## 🐛 Возможные проблемы

**Бот не запускается:**
- Проверьте правильность BOT_TOKEN в файле .env
- Убедитесь, что все зависимости установлены

**Не работают команды:**
- Проверьте, что бот добавлен в чат с нужными правами
- Попробуйте команду /start

**Ошибки базы данных:**
- Удалите файл database.db и перезапустите бота
